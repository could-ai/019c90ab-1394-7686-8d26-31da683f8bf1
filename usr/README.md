# Student Academic Performance Analytics

An interactive Flutter application for comprehensive student performance analysis and visualization.

## Features

### Phase 1: Data Strategy and Preparation
- Unified dataset integration from Student Information Systems
- Data cleaning and validation (missing values, duplicates, normalization)
- Structured data models for students, courses, and grades

### Phase 2: Exploratory Data Analysis
- Descriptive statistics (mean, median, standard deviation)
- Grade distribution histograms
- Correlation analysis between LMS usage and academic performance
- Interactive scatter plots for data exploration

### Phase 3: Interactive Visualization Dashboard
- Real-time performance metrics and KPIs
- Performance heatmap by department and grade ranges
- Drill-down functionality for individual student analysis
- At-risk student identification and highlighting

### Phase 4: Insights and Predictive Analytics
- Early warning system for at-risk students
- Department-level performance comparisons
- Predictive modeling for intervention recommendations
- Actionable educational policy insights

## Technical Implementation

- **Framework**: Flutter for cross-platform mobile/web deployment
- **Data Visualization**: FL Chart library for interactive charts
- **State Management**: Flutter's built-in state management
- **Data Models**: Structured classes for type-safe data handling
- **UI/UX**: Material Design with responsive layouts

## Getting Started

1. Ensure Flutter SDK is installed
2. Run `flutter pub get` to install dependencies
3. Run `flutter run` to start the application
4. Navigate through the four phases using the home screen
